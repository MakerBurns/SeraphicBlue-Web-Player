ÅüMemory Tree Saving

Memory trees are transparent blue tree-shaped save points that can be found in various dungeons and cities.
Players can save their game progress via Memory Trees.


ÅüCutscene Saving
There are many series of cutscenes in Seraphic Blue that make up the game's storyline, in which players have no control over the characters.
Therefore, in between a series of cutscenes, the game will prompt you to save the game's progress (by bringing up the save screen as shown above) before continuing with the cutscenes.


ÅüThings to note while saving
It is always wise to save the game before boss battles, and in between cutscenes.
You'll know that a boss battle is coming either when you see a Return Tree (a transparent green tree that teleports you back to the start of a dungeon), or a Resurrection Point (a large blue sphere. This will be explained in another article). This is so that you'll be well-prepared before fighting a boss.

Cutscenes tend to be long in Seraphic Blue, so when you have the chance to save in between cutscenes, it would be wise to do so.

Also, when saving, be careful not to accidentally overwrite save files. It is always wise to make new save files than overwrite existing ones.

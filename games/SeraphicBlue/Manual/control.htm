These are the main controls required to play Seraphic Blue:

Up（↑） ・Move characters up
・Move cursor up
・Switch between single and multiple targets (when using certain skills in battle)
・Increase value when eg. buying/selling items
etc...
Down（↓） ・Move characters down
・Move cursor down
・Switch between single and multiple targets (when using certain skills in battle)
・Decrease value when eg. buying/selling items
etc...
Left（←） ・Move characters left
・Move cursor left
・Move to previous page (when browsing through list of items)
etc...
Right（→） ・Move characters right
・Move cursor right
・Move to next page (when browsing through list of items)
etc...
Enter, Space, Z ・Select a particular choice
・Proceed through dialogues
・Call for airship on the World Map
・Call for airship to land (while airship is in flight)
etc...
Esc, X ・Exit from Menu
・Cancel a particular choice
・Enter the interiors of the airship (while airship is in flight)
・Cancel an action when in battle
etc...
Shift ・Generally used for special features, eg. Accessing Second Command in Battle
F4 ・Switch between fullscreen and windowed modes (Windows Function)
F5 ・Change the size of the game screen
F12 ・Go back to Title Screen (Reset)
Alt + F4 ・Exit from the game (Windows Function)
